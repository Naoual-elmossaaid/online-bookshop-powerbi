# Online Bookshop Analysis with Power BI

This project shows a Power BI dashboard for analyzing sales data of an online bookshop.

## Project Overview

The dashboard includes:

* Total revenue KPI
* Average price KPI
* Total sales KPI
* Top 5 books by revenue
* Sales by author country
* Rating vs revenue analysis
* Sales over time
* Sales by format (E-Book, Hardcover, Paperback)

The data model is built using a Star Schema with a central Sales table and dimension tables for books, authors, publishers and ratings.
